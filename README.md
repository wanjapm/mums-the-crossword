# Mums The Crossword

I wanted to make for my mum an app where she can play crossword puzzles from Old Awake! Magazines from [WOL](https://wol.jw.org)

Play the puzzles [here](https://wanjapm.github.io/mums-the-crossword/)

- Awake! Aug 8, 2004
